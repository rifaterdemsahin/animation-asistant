Go standard testing package — go test ./... with coverage and benchmarks.

Test Library

The backend (Go server/) uses Go's built-in testing package. Tests run with:

go test ./server/...           # all tests
go test -v ./server/...        # verbose output
go test -cover ./server/...    # with coverage
go test -run ^TestX$ ./server/ # single test
