ℹ️ About this comparison

Both providers receive the exact same system + user prompt (built from the project's Q&A, outline and storyboard context) and run in parallel. DeepSeek is queried via its OpenAI-compatible API (deepseek-chat) with JSON output mode enforced. Results are not persisted — this is a read-only A/B test. To actually save a script with a provider, use the 📜 Script Writer and pick the model in the "Script model" dropdown.

Configure: DEEPSEEK_API_KEY (comma-separated for rotation), DEEPSEEK_MODEL (default deepseek-chat), DEEPSEEK_BASE. Keys: platform.deepseek.com/api_keys.
