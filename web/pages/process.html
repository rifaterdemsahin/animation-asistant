🔄 Production Process

End-to-end asset production pipeline for Canva animation components. Built for self-learning through creation.

📝 1. Project — Question & Answer

Frame your topic as a Q&A pair. The question drives the narrative arc; the answer feeds the lesson.

🆕 Create Project 📁 Browse Projects

Example: "How does machine learning work?" → "It learns patterns from data to make predictions."

📋 2. Storyboard — 3 Infographics × 4 Frames

Generate the visual blueprint. Three infographic images — one per act — each divided into 4 frames in a 2×2 grid:

😱 Act 1: Problem
Frame 1: World setup
Frame 2: Pain point
Frame 3: Stakes
Frame 4: Tension
💡 Act 2: Solution
Frame 1: Key insight
Frame 2: Mechanism
Frame 3: Application
Frame 4: Validation
🎓 Act 3: Lesson
Frame 1: Recap
Frame 2: Impact
Frame 3: Takeaway
Frame 4: Call to action
📋 Open Storyboard

🎛️ 3. Media Manager — Generate Assets

Produce all animation components per act. The pipeline runs sequentially:

📝 Outline → 📜 Script → 🖼️ Components → 🎙️ Voiceover → 🎵 Music → 🔉 SFX

Each step generates per-act outputs saved to Azure Blob Storage. All prompts are audited. All assets are prefixed with the project slug for Canva compatibility.

🎛️ Media Manager ⚡ Bulk: Run Full Pipeline
🖼️ Typed images: background, lower-third, speech-bubble, infographic — 4 types × 3 acts = 12 images
🎙️ Narration MP3: ElevenLabs George voice
🎵 Background music: fal.ai cinematic loops
🔉 Sound effects: whoosh, ding, reveal per act

📥 4. Download Assets → Place in Canva

All generated files are browsable and downloadable from the 🔽 Generated Files panel in Media Manager. Each asset carries a direct URL.

📁 Browse Files

View images + audio in Media Manager, download individually.

☁️ Azure Storage

Direct blob access for bulk download via Azure Portal.

🎨 5. Canva — Self-Learning Loop

Place generated components into Canva video timelines. The act of arranging and editing in Canva creates an aesthetic learning experience — the core self-learning mechanism of this project.

🎨 Open Canva Paste your Canva project link in the project metadata.
